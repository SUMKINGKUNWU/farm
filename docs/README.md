# Farm Exchange 文档库

本目录用于沉淀农场牧场经营类金融游戏的产品、技术、版本与落地方案。

## 文档原则

- 所有需求分析、设计决策、开发完成记录都必须落到本地文档。
- 每完成一个阶段，只新增一份版本文档，不修改旧版本文档。
- 旧文档代表当时的决策快照，即使后续方案变化，也通过新文档增量说明。
- 代码实现、数据库设计、接口设计、原型调整都需要在对应版本文档中记录。

## 推荐目录

- `docs/versions/`：每次阶段完成后的增量版本文档。
- `docs/reference/`：相对稳定的参考文档、术语表、规则草案。

## 当前技术栈

- 前端：Vue 3 + Vite + Pinia + ECharts
- 后端：Spring Boot
- 数据库：PostgreSQL
- 缓存与实时辅助：Redis
