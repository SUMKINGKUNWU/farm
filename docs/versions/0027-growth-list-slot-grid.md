# 0027 生产批次列表与 16 格地块

## 当前已有能力

- 玩家端已经具备注册登录、概览、交易密码、商店购买、扩建、生产、收获、交易站买卖。
- 后端已有生产开始、收获、库存查询能力。
- 前端统一错误处理已接入玩家端和管理台。

## 本次完成内容

- 后端新增 `GrowthInstanceResponse`，用于返回玩家生产批次明细。
- 后端新增生产批次列表接口：
  - `GET /api/users/{userId}/growth`
  - `GET /api/me/growth`
- 生产批次列表返回：
  - 批次 ID
  - 栏位类型
  - 栏位 ID
  - 投入商品编码
  - 产出商品编码
  - 产出数量
  - 开始时间
  - 成熟时间
  - 收获时间
  - 状态
- 玩家端 dashboard 刷新时同步加载生产批次。
- 玩家端农田和牧场改为最多 16 格可视化地块。
- 地块支持展示：
  - 未扩建
  - 空闲
  - 生产中
  - 可收获
- 玩家可直接点击空闲农田/牧场格子开始种植或养殖。
- 玩家可直接点击生产中且已成熟的格子收获。
- 玩家端新增生产批次表格，便于查看所有批次和手动收获。

## 风险与说明

- 目前地块状态来源于栏位列表和生产批次列表的前端组合，后端尚未提供“栏位 + 当前批次”的聚合接口。
- 可收获判断由前端根据 `readyAt` 与本机时间判断，后端仍会在收获时做最终校验。
- 生产批次列表当前包含历史已收获批次，后续可增加状态筛选或分页。
