# 0029 前端视觉原型

## 当前已有能力

- 玩家端和管理台功能已完成 MVP。
- 玩家端包含农场、牧场、商店、交易站、私下交易、批次列表、16 格地块。
- 当前前端可用，但视觉偏功能直白，游戏感和农场氛围不足。

## 本次完成内容

- 新增 5 张本地 SVG 视觉原型，先用于确定最终样式，不直接改现有 Vue 页面。
- 原型 A：`ui-style-a-woodland-board.svg`
  - 木牌农场看板方向，游戏棋盘感强。
- 原型 B：`ui-style-b-cozy-desk.svg`
  - 暖色经营桌面方向，资源和任务更集中。
- 原型 C：`ui-style-c-market-town.svg`
  - 集市交易镇方向，更强调交易站、报价、税费。
- 原型 D：`ui-style-d-soft-mobile-farm.svg`
  - 参考小页签图片后的柔和玻璃拟态农场主界面。
  - 大圆角、浅灰背景、底部悬浮导航、中央加号、少文字。
- 原型 E：`ui-style-e-soft-market-trade.svg`
  - 参考小页签图片后的柔和交易站界面。
  - 以行情曲线、买卖卡片、私下报价、底部胶囊主按钮为核心。

## 参考方向吸收

- 大圆角主容器。
- 白色/浅灰玻璃拟态卡片。
- 内高光和柔和阴影。
- 底部悬浮页签托盘。
- 中央突出主操作按钮。
- 少文字、多图标、多数字反馈。
- 页面整体更像游戏 UI，而不是后台表格。

## 验证结果

- 本次为 SVG 原型文件，无需运行前后端构建。
- SVG 已落地到本地 `prototypes/farm-exchange/` 目录。

## 下次建议目标

- 由原型 D/E 作为主方向，落地到 Vue 页面。
- 将现有玩家端拆成更少文字的主界面：农场、商店、交易站、私下交易用底部悬浮页签切换。
- 管理台可保留较强数据面板风格，但同步改成柔和卡片系统。
