# 0031 常规比例悬浮玻璃信息原型

## 当前已有内容

- 已有 `0030` 紧凑玻璃拟态探索原型。
- 用户反馈：不要紧凑型，之前只是说明个人信息、物品信息可以借鉴玻璃拟态做悬浮展示。

## 本次完成内容

- 新增常规比例原型：`prototypes/farm-exchange/ui-style-g-standard-floating-glass.svg`。
- 撤回“紧凑手机比例”作为落地方向的判断，不删除旧文件，仅作为探索记录保留。
- 主界面恢复为正常网页游戏尺度：
  - 大尺寸农田与牧场操作区。
  - 右侧交易站行情和交易入口。
  - 底部悬浮页签继续参考小页签质感。
- 玻璃拟态只用于悬浮信息层：
  - 点击头像后的个人信息悬浮卡。
  - 点击地块或商品后的物品信息悬浮卡。
  - 点击交易后的交易确认悬浮卡。
- 主玩法区域使用纸感、草地、土地、金币、深绿色交易板等稳定材质，避免主界面过度透明。

## 当前设计结论

- 主界面不做紧凑手机 HUD，也不做全局玻璃拟态。
- 页面应是网页端游戏主界面比例，信息密度可以适中，但视觉上要有农场地图感。
- 玻璃拟态仅作为局部浮层风格，用于临时展示和确认，不承担主容器。

## 下次建议

- 若用户确认 G 版方向，下一步将 Vue 玩家端按该结构改造。
- 优先落地：
  - 农田/牧场主区域。
  - 右侧交易站卡片。
  - 底部悬浮页签。
  - 头像、物品、交易确认 3 类玻璃悬浮层。
