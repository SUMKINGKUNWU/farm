# 0043 管理台子面板拆分

## 当前已有

- 前端已形成玩家端游戏化工作区和管理端工作区两个主要页面。
- 管理端支持税率配置、大宗交易令牌发放、玩家资产查看、交易记录与图表查看。
- 通用提示块、统计卡片已抽为公共组件。
- ECharts 已改为动态导入，避免进入首屏主包。

## 本次完成

- 将管理台从单个大组件拆分为 4 个子面板：
  - `TaxConfigPanel.vue`：税率读取、编辑和保存。
  - `BulkTokenPanel.vue`：大宗交易令牌发放与最近发放结果展示。
  - `PlayerAssetsPanel.vue`：玩家资产摘要与库存表格。
  - `TradeRecordsPanel.vue`：交易图表和交易记录表格。
- `AdminWorkspace.vue` 调整为组合容器，只负责管理台顶部说明、状态提示和子面板装配。
- 修复拆分过程中出现的中文编码异常和模板标签破损，确保页面可正常编译。
- 保留交易记录图表的 ECharts 动态加载逻辑。

## 验证结果

- 已执行 `npm.cmd run build`，前端生产构建通过。
- 构建仍提示 ECharts chunk 超过 500 kB；该 chunk 已由动态导入隔离，不影响主入口体积。

## 下次建议

- 继续拆分玩家端工作区中的农场、牧场、交易站、私下交易和悬浮信息面板。
- 拆分完成后再做一次前端交互回归，重点验证移动端布局、键盘可达性和交易密码前置流程。
