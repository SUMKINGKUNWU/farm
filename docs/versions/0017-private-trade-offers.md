# 0017 - 私下交易报价单

## 当前已有

当前项目已经具备：

- 用户注册与初始资产。
- 交易密码设置与校验。
- 商店购买。
- 种植、养殖、收获。
- 交易站即时买卖。
- 交易站行情和价格快照。

## 本次完成

本次实现私下交易 MVP：

- 卖方创建报价单。
- 创建报价单时冻结卖方库存。
- 卖方取消报价单。
- 取消报价单时释放冻结库存。
- 买方接受报价单。
- 买方接受时校验交易密码。
- 买方支付交易金额和 5% 私下交易税。
- 卖方获得交易金额。
- 商品从卖方转移到买方。
- 写入税收记录。
- 写入金币和商品资产流水。
- 防止重复接受已完成报价单。

## 新增接口

### 创建报价单

```text
POST /api/users/{sellerUserId}/private-trades
```

请求：

```json
{
  "buyerUserId": "...",
  "itemCode": "WHEAT",
  "quantity": 20,
  "priceAmount": 1000,
  "tradePassword": "654321"
}
```

规则：

- 卖方必须输入交易密码。
- 买方必须存在且状态可用。
- 卖方不能和自己交易。
- 当前 MVP 仅支持收获物交易。
- 卖方库存必须足够。
- 创建后冻结卖方库存。
- 报价单有效期当前为 24 小时。

### 接受报价单

```text
POST /api/users/{buyerUserId}/private-trades/{offerId}/accept
```

请求：

```json
{
  "tradePassword": "654321"
}
```

规则：

- 只有指定买方可以接受。
- 买方必须输入交易密码。
- 报价单必须是 `WAIT_ACCEPT`。
- 报价单不能过期。
- 买方金币必须足够支付 `priceAmount + taxAmount`。
- 卖方获得 `priceAmount`。
- 系统记录 `taxAmount`。
- 冻结库存转移给买方。

### 取消报价单

```text
POST /api/users/{sellerUserId}/private-trades/{offerId}/cancel
```

规则：

- 只有卖方可以取消。
- 只有 `WAIT_ACCEPT` 状态可以取消。
- 取消后释放冻结库存。

## 税费规则

私下交易税率来自：

```text
tax_config.PRIVATE
```

当前默认：

```text
500 基点 = 5%
```

计算：

```text
taxAmount = priceAmount * 500 / 10000
buyerCost = priceAmount + taxAmount
sellerIncome = priceAmount
```

## 事务一致性

创建报价单在同一事务中完成：

```text
校验卖方交易密码
校验买方状态
校验商品
冻结卖方库存
创建报价单
写冻结资产流水
```

接受报价单在同一事务中完成：

```text
校验买方交易密码
锁定报价单
锁定买卖双方钱包
校验状态和过期时间
扣买方金币
加卖方金币
扣卖方冻结库存
加买方库存
更新报价单完成
写税收记录
写资产流水
```

取消报价单在同一事务中完成：

```text
锁定报价单
校验卖方身份和状态
释放冻结库存
更新报价单取消
写释放资产流水
```
