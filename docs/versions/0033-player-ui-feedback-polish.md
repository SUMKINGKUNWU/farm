  - `Open`：可投入生产。
  - `Growing`：成长中。
  - `Ready`：可收获。
- 农田和牧场栏位的操作提示更明确：
  - 空闲农田显示 `Plant`。
  - 空闲牧场显示 `Raise`。
  - 可收获批次显示 `Click harvest`。
  - 未扩建栏位显示 `Expand first`。
- 底部悬浮页签增加小图标区域，让导航更接近游戏式页签。
- 玻璃悬浮卡增加提示块：
  - 个人信息卡说明交易密码用途。
  - 物品信息卡说明后续可承载价格、成长、库存和用途。
  - 交易确认卡说明后端仍会校验余额、库存、交易密码和大宗令牌。
